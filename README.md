# Sauce Demo pytest framework

Pure pytest + Selenium framework for Sauce Demo UI using POM.

## Setup

1) Create and activate a virtual environment.
2) Install dependencies:

```bash
pip install -r requirements.txt
```

## Run tests

```bash
pytest
```

By default tests run with 2 workers (see `pytest.ini`).

Override worker count (2 workers):

```bash
pytest -n 2
```

Parallel run (auto workers):

```bash
pytest -n auto
```

## Reports and screenshots

Allure results are generated by default in `reports/allure-results`.
Failure screenshots are saved in `reports/screenshots/` and attached to Allure.

To view the Allure report (requires Allure CLI installed):

```bash
allure serve reports/allure-results
```

## Configuration

Environment variables (optional):
- BASE_URL (default https://www.saucedemo.com/)
- STANDARD_USER (default standard_user)
- STANDARD_PASSWORD (default secret_sauce)
- PAGE_LOAD_TIMEOUT (default 20)
- IMPLICIT_WAIT (default 0)
- EXPLICIT_WAIT (default 10)
